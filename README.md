## Installation

- Clone the repository
- Run:

```sh
$ npm install
```
- Start a mongodb instance locally:

```sh
$ mongod
```
- Put the mongo uri on an .env file like in the example

## Run

```sh
$ npm start
```

## Create an admin user

- Create a regular user
- Change its admin attribute to true
